# python-catch-the-turtle
A simple Catch The Turtle game built with Python Turtle using random movement and timers.

This is a simple "Catch The Turtle" game developed using Python's Turtle module.

## Game Rules
- A turtle appears at a random position on the screen.
- The turtle stays visible for 3 seconds.
- If the player clicks the turtle, the score increases.
- If the turtle is not clicked in time, it counts as a miss.
- The game lasts 30 seconds.

## Features
- Random turtle movement (no grid system)
- Click detection using events
- Score and miss counter
- Countdown timer
- Game over screen
